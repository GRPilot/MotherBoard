#IEEE 1394
{ (FireWire, i-Link) - последовательная высокоскоростная шина, предназначенная для обмена цифровой информацией между компьютером и другими электронными устройствами. }

#Audio
{ Разъем аудио }

#Питание CMOS.
{ CMOS (Complementaty Metal-Oxide Semiconductor, комплиментарный металло-оксидный полупроводник (КМОП)) - технология изготовления схем, используемая  для хранения  некоторых параметров компьютера при отключении его блока питания. 
Микросхема CMOS памяти использует в качестве источника питания - батарею. }

#DDR2 SDRAM 
{ (Double-Data-Rate Two Synchronous Dynamic Random Access Memory — синхронная динамическая память с произвольным доступом и удвоенной скоростью передачи данных, второе поколение) — это тип оперативной памяти используемой в вычислительной технике в качестве оперативной и видеопамяти.
Внешне - 240 контактов (по 120 с каждой стороны). }

#Super I/O
{ (контроллер ввода/вывода) - объединяет интерфейсы различных низкочастотных устройств. 
Включает в себя следующие функции:
1. контроллер дисковода флоппи;
2. параллельный порт (LPT) (обычно используется для принтеров);
3.один или более последовательных (COM) портов;
4. интерфейс мыши;
5. интерфейс клавиатуры.}

#Разъем локальной сети.
{ LAN (LOCAL AREA NETWORK, локальная вычислительная сеть) - сеть, ограниченная пределами определенной территории. }

#Материнская плата
{ (англ. motherboard) — это сложная многослойная печатная плата, на которой устанавливаются основные компоненты персонального компьютера (центральный процессор, контроллер ОЗУ и собственно ОЗУ, загрузочное ПЗУ, контроллеры базовых интерфейсов ввода-вывода). 
Материнская плата содержит разъёмы (слоты) для подключения дополнительных контроллеров, для подключения которых обычно используются шины USB, PCI и PCI-Express.
Card (плата) -карта, вставяемая в разъем шины. }

#CHIPSET (чипсет) 
{ Набор интегральных схем, образующих функциональный блок.
Состоит из«северного» и «южного мостов».
Bridge (мост) - устройство, предназначенное для соединения нескольких шин (сетей) в одну общую шину (сеть) с передачей данных в режиме разделения времени.}

#MCH (Memory Controller Hub) 
{Контроллер-концентратор памяти— северный мост (англ. northbridge). Обеспечивает взаимодействие центрального процессора (ЦП) с памятью и видеоадаптером (PCI Express). }

#ICH (I/O Controller Hub)
{ Контроллер-концентратор ввода-вывода — южный мост (англ. southbridge). Обеспечивает взаимодействие между ЦП и жестким диском, картами PCI, интерфейсами IDE, SATA, USB и пр. }

#Разъем PATA (IDE).
{ (Parallel Advanced Technology Attachment) - параллельная шина (интерфейс) жесткого диска. }


#IDE
{(Integrated Drive Electronics) - параллельный интерфейс подключения накопителей (жёстких дисков и оптических приводов).}

#Слоты расширения PCI: плата LAN, плата SCSI, плата или хост USB.
{ PCI (Periphecal Component Interconnect) - соединение периферийных компонентов, параллельная мультиплексная шина общего назначения с пакетной передачей данных.}

#Разъем PCI-Ex16 (графическая карта).
{PCI-Express (Periphecal Component Interconnect Express) - последовательная пакетная дифференциальная шина с возможностью наращивания числа каналов.
Графическая плата(графическая карта, видеоадаптер(videocard)) -  устройство, преобразующее изображение, находящееся в памяти компьютера, в видеосигнал для монитора.}

#Разъемы SATA.
{SATA (Serial Advanced Technologe Attanchment) - последовательная дифференциальная универсальная шина (интерфейс) жесткого диска.}

#Сокет процессора.
{SOCKET (сокет) - гнездо, в которое устанавливается микросхема со штырьковыми выводами.}

#Разъемы USB.
{USB (Universal Serial Bus) - последовательная дифференциальная универсальная шина, для подключения различных периферийных устройств.}

#Разъем VGA(DVI) монитора.
{ VGA (VIDEO GRAFICS ARRAY, видеографическая решетка) - видеоадаптер, доддерживающий текстовый и графический режимы.}
